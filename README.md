# ResumeIQ — NLP Resume Analyzer

ResumeIQ is an AI-powered Resume Analyzer that matches resumes with job descriptions using NLP and Machine Learning techniques.

## Features
- SBERT Semantic Similarity
- TF-IDF Keyword Analysis
- ATS Resume Scoring
- Skill Matching System
- PDF Resume Parsing
- AI-based Resume Suggestions
- Streamlit Web Interface

## Tech Stack
- Python
- NLP
- Sentence-BERT
- Streamlit
- spaCy
- Scikit-learn
- PDFPlumber

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
